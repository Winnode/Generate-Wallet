## Installation
manual Install 
- open cmd 
- cd generate-wallet  
- nmp install bip39
- nmp install @solana/web3.js
- nmp install eth-lib
- nmp install readline-sync
- nmp install xlsx
- nmp install ethereum-mnemonic-privatekey-utils
- nmp install ethers
