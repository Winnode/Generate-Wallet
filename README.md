## Installation
manual Install 
- open cmd 
- cd generate-wallet  
- npm install bip39
- npm install @solana/web3.js
- npm install eth-lib
- npm install readline-sync
- npm install xlsx
- npm install ethereum-mnemonic-privatekey-utils
- npm install ethers
